The file "checkpartial_contest_calls_random.txt" contains more than 37.000 call signs, that had appeared in major contest logs over the years, taken from checkpartial.com.
The file is randomized (i.e. not in alphabetical order).

To practice call signs with Morserino-32:
- upload the file to Morserino-32 with WLAN
- set the Parameter "Radmoniz File to "On"
- play it with CW generator/File Player or Echo Trainer / File Player
